# Geo-Energy Solutions Website

Έτοιμο Next.js + Tailwind project για ανέβασμα στο Vercel.

## Τρόπος 1: Vercel χωρίς πολλά τεχνικά
1. Κάνε unzip αυτόν τον φάκελο.
2. Πήγαινε στο https://vercel.com
3. Κάνε login.
4. Πάτα Add New Project.
5. Ανέβασε/σύνδεσε τον φάκελο project.
6. Deploy.

## Τρόπος 2: GitHub + Vercel
1. Κάνε unzip.
2. Ανέβασε όλα τα αρχεία σε GitHub repository.
3. Στο Vercel πάτα Add New Project.
4. Διάλεξε το GitHub repository.
5. Πάτα Deploy.

## Τοπική δοκιμή
```bash
npm install
npm run dev
```
Άνοιγμα: http://localhost:3000
