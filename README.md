# Geo-Energy Solutions

Έτοιμο Next.js project για Vercel.

## Περιεχόμενα
- app/page.tsx
- app/layout.tsx
- app/globals.css
- package.json
- postcss.config.mjs

## Ανέβασμα στο GitHub
1. Κάνε unzip.
2. Ανέβασε ΟΛΑ τα περιεχόμενα του φακέλου στο GitHub repo.
3. Κάνε commit.
4. Στο Vercel πάτα Redeploy.

## Σημαντικό
Μην ανεβάσεις το zip ως zip στο GitHub. Ανέβασε τα αρχεία που βρίσκονται μέσα στον φάκελο.
